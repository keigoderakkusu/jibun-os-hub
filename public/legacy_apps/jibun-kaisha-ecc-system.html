概要 / OVERVIEW

🎯 ECC を自分株式会社の「脳」として統合する戦略

everything-claude-code（ECC）は、Anthropicハッカソン優勝者が10ヶ月かけて構築したエージェントOSです。 Skills（スキル）・Hooks（自動フック）・Agents（特化エージェント）・Memory（記憶継続）の4レイヤーが揃っており、 KeigoさんのGAS自動化パイプラインと組み合わせることで、コーディング・リサーチ・コンテンツ生成・副収入獲得を ほぼノーコードで自動化できます。Claude Code を「チームメンバー」として使う発想に切り替えると、 自分株式会社のCEOとしてClaude Codeが24時間稼働する社員になります。

セットアップ手順 / INSTALL FLOW
1

Claude Code に ECC をインストール

ターミナルで以下を実行。/plugin marketplace add affaan-m/everything-claude-code → /plugin install everything-claude-code

2

jibun-kaisha リポジトリに CLAUDE.md を作成

プロジェクトのルートに自分株式会社の文脈（エージェント構成・GAS構造・副業目標）を記述した CLAUDE.md を置く。ECCがこれを毎回自動参照する。

3

副業ごとにスキルを有効化

アフィリエイト → article-writing / content-engine を有効化。フリーランス営業 → investor-outreach を有効化。必要なスキルだけ選択的にインストール可能。

4

NanoClaw で記憶を永続化

セッション開始時に /loop-start を実行。作業終了時に /quality-gate で成果物チェック＆記憶保存。次回セッションから Keigo さんの文脈が引き継がれる。

5

コスト最適化（重要）

ECC のトークン最適化ガイドに従い、CLAUDE_CODE_SUBAGENT_MODEL=haiku で サブエージェントをHaikuに。探索・リサーチ系タスクのコストを最大80%削減できる。

コスト最適化 TIPS
モデル戦略
Sonnet → Opus
80%のタスクはSonnetで十分。複雑な推論時だけ /model opus に切替えてコスト60%削減
サブエージェント
Haiku 固定
ファイル読み・テスト実行はHaikuに委譲。メインに比べ80%安く、品質は十分
Auto-compact
50% に設定
デフォルト95%では遅すぎる。50%でコンパクトしてセッション品質を維持
Think tokens
MAX=10,000
拡張思考のデフォルト上限を制限。単純タスクは0に。隠れコストを70%削減

# ── ECC インストール（Claude Code 必須）──────────────────
gh ext install affaan-m/everything-claude-code
