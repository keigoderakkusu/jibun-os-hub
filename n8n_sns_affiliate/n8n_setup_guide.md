# 🛠 n8n_setup_guide: 実装手順書

このドキュメントでは、準備段階からn8nへのデプロイまで、ステップバイステップで解説します。

---

## 🔑 ステップ1: APIキーの取得

### OpenAI API キーの取得
1. [OpenAI Developer Platform](https://platform.openai.com/) にアクセスし、ログイン（または新規登録）します。
2. 左側メニューの「API keys」をクリックします。
3. 「Create new secret key」をクリックし、名前（例: `n8n_sns`）を付けてキーを生成します。
4. **表示されたキー（`sk-...`）をコピーして、メモ帳などに保存してください**（二度と表示されません）。
5. ※重要: OpenAIは従量課金制です。Settings > Billingからクレジットカード情報を登録し、$5〜$10程度チャージしておいてください。

### Google Sheets の準備
1. Google Driveを開き、新しいスプレッドシートを作成します。
2. シートの名前を「SNS投稿案」などにします。
3. 1行目にヘッダーを作成します：
   - A1: `Date`
   - B1: `Title`
   - C1: `URL`
   - D1: `Post_Draft`
4. URLの中から、**スプレッドシートID**をコピーしてメモします。
   （例: `https://docs.google.com/spreadsheets/d/ココがIDです/edit`）

---

## 🚀 ステップ2: n8nのセットアップ

### 1. ワークフローのインポート
以下のどちらかの方法で `n8n_workflow.json` をn8nに読み込ませます。
- **ドラッグ＆ドロップ**: n8nの画面にJSONファイルを投げ込む。
- **メニューからインポート**: 右上の「...」（またはメニュー）>「Import from File」を選択し、JSONファイルを選択。

### 2. Credentials（認証情報）の設定
インポートすると、各ノードにエラーマーク（⚠️）がついているはずです。認証情報を設定しましょう。

#### OpenAIノードの設定
1. OpenAIノードをダブルクリック。
2. `Credential for OpenAI` のプルダウンから `- Create New -` を選択。
3. 名前（例: `ChatGPT SNS`）を付け、先ほど取得したAPIキー（`sk-...`）を貼り付け保存（Save）。

#### Google Sheetsノードの設定
1. Google Sheetsノードをダブルクリック。
2. `Credential to connect with` で `Google Sheets OAuth2 API` 等を選択し `- Create New -`。
3. `Sign in with Google` を押し、ご自身のGoogleアカウントで認証を完了させます。
4. `Document ID` フィールドに、先ほどメモしたスプレッドシートIDを選択・入力します。

---

## ✅ ステップ3: テスト実行と動作確認

1. すべての設定が終わったら、画面下部の **「Execute Workflow」** ボタンをクリックします。
2. ノードが順番に緑色になり、データが流れるのを確認します。
3. スプレッドシートを開き、新しい行にAIが生成した投稿文が追記されていれば**成功です！**

---

## 🔄 ステップ4: 日々の運用

1. ワークフロー画面右上のスイッチを **「Active」** に切り替えます。
   （これでスケジュール通りに自動実行されるようになります）
2. **毎日のルーティン（10〜20分）**:
   - スプレッドシート（または出力先）をチェック。
   - 生成された投稿文を読み、あなたの感想や意見を追記。
   - アフィリエイトリンク（証券口座やツールの紹介リンク）を適宜挿入。
   - X（旧Twitter）に投稿（または予約投稿ツールに入れる）。
3. 継続して反応（いいね・リポスト）を見ながら、次の `n8n_prompt_guide.md` を活用してAIへの指示（プロンプト）を改善していきましょう。
