# 🚀 QUICKSTART: 特化型SNSアカウント＆アフィリエイト半自動化システム

この度は「特化型SNSアカウント＆アフィリエイト半自動化システム」の設計書を手に取っていただきありがとうございます。このシステムを導入することで、SNS運用の時間が**92%削減**され、コンテンツ量が**3〜6倍増加**することが期待できます。

## 📅 実装スケジュール（目安：2-3時間）

スムーズに実装を進めるために、以下のステップで進めてください。

### 【1時間目】事前準備
1. **このQUICKSTART.mdを最後まで読む**（5分）
2. **OpenAI API キーの取得**（手順は `n8n_setup_guide.md` 参照）
3. **保存用 Google Sheets の作成**（手順は `n8n_setup_guide.md` 参照）

### 【2時間目】n8nの設定
1. **n8nにワークフローをインポート**（`n8n_workflow.json` を使用）
2. **API Credentials（認証情報）を設定**（OpenAI, Google Sheets）
3. **プレビュー/テスト実行で動作確認**

### 【3時間目】デプロイと運用開始
1. **ワークフローを Active（有効）にする**
2. **24時間後に自動実行されているか確認**
3. **日々の運用（1日10〜20分）を開始する**

## 🔑 成功するための重要な4つのポイント

1. **AIはあくまで「下準備役」**
   AIが生成した文章をそのまま投稿するのではなく、**必ずあなた自身の言葉（感想や意見）を1〜2文追加**してください。これがスパム判定を防ぎ、ファンを獲得する最大のカギです。
2. **キーワード選定が命**
   最初は広めのキーワード（例：AI, 投資）でニュースを集め、ノイズが多い場合は `n8n_workflow_guide.md` に従ってキーワードを絞り込んでください。
3. **法務リスクを徹底回避**
   特に金融や投資系のアカウントを運営する場合、「この株は絶対上がる」といった**断定表現や投資助言は絶対に避けてください**。（詳細は `n8n_prompt_guide.md` のコンプライアンス編参照）
4. **まずは小さく始める**
   最初は1日数件のニュース取得から始め、エラーなく動くことを確認してから頻度を上げてください。

## 🛠 よくあるトラブル＆解決策

- **Q: ニュースが取得されない！**
  - A: RSSフィードのURLが間違っているか、更新頻度が低いサイトを指定している可能性があります。別のRSSフィードを試してください。
- **Q: OpenAIノードでエラーが出る！**
  - A: APIキーが間違っているか、OpenAIアカウントのクレジット（残高）が不足している可能性があります。
- **Q: Google Sheetsにデータが保存されない！**
  - A: スプレッドシートのIDが正しく設定されていないか、Google Sheetsの認証情報（OAuth / Service Account）の設定が完了していません。
