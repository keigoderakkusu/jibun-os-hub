## 🚀 セットアップから実行まで

### ステップ 1: GAS バックエンド（共通）- 5 分

```bash
1. Google Apps Script (script.google.com) を開く
2. 新規プロジェクト: "DocumentScanSystem"
3. DocumentScanAdvancedBackend.gs をコピー＆貼付
4. デプロイ → ウェブアプリ → 「アクセス: 全員」
5. デプロイメント URL をコピー
   ✅ 完了: https://script.google.com/macros/d/abc123xyz789/usercallback
```

### ステップ 2: モバイルカメラ版（オプション）- 3 分

```bash
1. DocumentScanMobile.gs を GAS に追加
2. 同じくウェブアプリとしてデプロイ
3. スマートフォンでデプロイメント URL を開く
   ✅ 完了: カメラアプリが起動
```

### ステップ 3: 自動スクロール版（オプション）- 5 分

```bash
1. Node.js v14+ をインストール
2. npm install puppeteer dotenv
3. .env ファイルに GAS_DEPLOYMENT_URL を設定
4. コマンド実行:
   node auto-scroll-screenshot.js --url "https://example.com"
   ✅ 完了: Google Drive に PDF が保存される
```

### ステップ 4: Chrome 拡張機能（オプション）- 2 分

```bash
1. 4 つのファイル（manifest.json, popup.html, popup.js, background.js）を用意
2. GAS URL を設定
3. Chrome に読み込む
   ✅ 完了: ツールバーに拡張機能アイコンが表示
```

**最短セットアップ時間: 5 分（GAS バックエンドのみ）**

## 💡 使用例（実績）

### 例 1: 営業報告書を毎日自動作成

```bash
# Google Cloud Scheduler で毎朝 9:00 実行
node auto-scroll-screenshot.js \
  --url "https://sales.example.com/report" \
  --scroll-mode "viewport" \
  --filename "daily-report"

→ 毎朝 Google Drive に営業レポート PDF が自動保存
→ チーム全員が確認可能
```

### 例 2: 顧客との契約書を即座にスキャン

```
1. 契約書を渡される
2. スマートフォンでデプロイメント URL を開く
3. カメラで契約書の全ページを撮影（複数ページモード）
4. 「💾 保存」をクリック
5. 即座に Google Drive に PDF が保存（全ページ分）

→ 紙の管理が不要に
→ 検索・共有が容易に
```

### 例 3: ブログ記事をオフライン保存

```bash
# Zenn の記事を PDF 化
node auto-scroll-screenshot.js \
  --url "https://zenn.dev/example/articles/abc123" \
  --scroll-mode "viewport" \
  --filename "zenn_article"

→ ブラウザオフラインでも読める PDF に変換
→ 長期保存が可能
```

### 例 4: EC サイト商品情報を全キャプチャ

```bash
# Amazon の商品リストを複数ページスクショ
node auto-scroll-screenshot.js \
  --url "https://amazon.com/s?k=nodejs" \
  --scroll-mode "paginated" \
  --pdf-mode "split"

→ 各ページが個別 PDF に
→ 比較検討用資料に
```
